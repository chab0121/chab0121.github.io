5 HTML Tags I Learned

<header>
Defines the header area of a page or section.
<nav>
Contains the main navigation links of the website.
<main>
Holds the main unique content of the page.
<section>
Groups related content together in a meaningful section.
<img>
Displays an image (use src and alt attributes).
